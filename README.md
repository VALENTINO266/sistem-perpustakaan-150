# Perpustakaan Digital SDN 150 OKU

## Deskripsi Singkat
Perpustakaan SDN 150 OKU adalah aplikasi manajemen perpustakaan berbasis web yang dirancang untuk mendigitalisasi sirkulasi buku secara praktis dan efektif. Aplikasi ini menyediakan kemudahan bagi petugas dalam mengelola stok buku serta data keanggotaan siswa guna meningkatkan efisiensi layanan literasi di sekolah.

### Fitur Utama
- Dashboard: Menampilkan total buku, siswa, peminjaman aktif, serta grafik sirkulasi buku harian secara visual.
- Buku: Menambah, mengubah, menghapus data katalog buku, serta memantau sisa stok buku di rak.
- Siswa: Mengelola data keanggotaan dan profil siswa, serta melacak riwayat keaktifan membaca setiap siswa.
- Pinjam dan Kembali: Mencatat transaksi pinjam-kembali buku secara digital dan menghitung batas waktu serta denda keterlambatan otomatis.

## Teknologi yang Digunakan
- Frontend: Next.js 
- Backend: Node.js 
- Database: PostgreSQL

## Cara Menjalankan Aplikasi
1. Clone Repositori
   git clone https://github.com/VALENTINO266/sistem-perpustakaan-150.git
   cd sistem-perpustakaan-150

2. Instalasi Dependensi
  npm install

3. Menjalankan Aplikasi
   npm run dev
